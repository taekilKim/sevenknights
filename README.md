# sevenknights

## 스킬 작성 가이드

### 자동 스타일링 박스 조건

스킬 설명을 작성할 때, 특정 키워드를 포함하면 자동으로 스타일링 박스가 적용됩니다.

#### 스킬 효과 박스
- `[스킬 강화]` - 스킬 강화 효과 박스 (노란색)
- `[6초월]` - 6초월 효과 박스 (보라색)
- `[2초월]` - 2초월 효과 박스 (파란색)

#### 대상 타겟 박스
다음 키워드가 포함된 문장은 자동으로 대상 타겟 박스가 적용됩니다:

**적군 다중 대상** (빨간색 박스)
- 모든 적군
- 전체 적군
- 적군 N명

**적군 단일 대상** (주황색 박스)
- 단일 적군
- 공격력이 가장 높은/낮은 적군
- 방어력이 가장 높은/낮은 적군
- 생명력이 가장 높은/낮은 적군
- 체력이 가장 높은/낮은 적군
- 속공이 가장 높은/낮은 적군
- 무작위 적군 1명
- 대상 적군
- 적군 1명

**아군 다중 대상** (파란색 박스)
- 모든 아군
- 전체 아군
- 아군 N명
- (스탯)이 가장 높은/낮은 아군 N명

**아군 단일 대상** (초록색 박스)
- 자신
- 단일 아군
- 공격력이 가장 높은/낮은 아군
- 방어력이 가장 높은/낮은 아군
- 생명력이 가장 높은/낮은 아군
- 체력이 가장 높은/낮은 아군
- 속공이 가장 높은/낮은 아군
- 대상 아군
- 아군 1명

#### 우선순위
1. 스킬 효과 박스 (`[스킬 강화]`, `[6초월]`, `[2초월]`)가 가장 먼저 적용됩니다
2. 대상 타겟 박스는 적군 다중 → 적군 단일 → 아군 다중 → 아군 단일 순서로 우선순위가 적용됩니다

#### 예시
```
모든 적군에게 피해를 입힙니다.
→ 적군 다중 대상 박스 적용

자신의 공격력을 증가시킵니다.
→ 아군 단일 대상 박스 적용

[스킬 강화] 피해량이 증가합니다.
→ 스킬 강화 효과 박스 적용
```
